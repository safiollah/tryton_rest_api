.. _model-ir.lang:

Language
========

The *Language* defines the properties such as the direction and the formatting
of the supported languages.
By loading the translations for a *Language*, it becomes translatable and can
be used to translate the user interface.

.. seealso::

   The Languages can be found by opening the main menu item:

      |Administration --> Localization --> Languages|__
